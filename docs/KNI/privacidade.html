Politica de Privacidade

Ultima atualizacao: 2 de marco de 2026.

1. Controlador de dados

Knimodas e a controladora dos dados pessoais tratados nesta plataforma para finalidade comercial, operacional, fiscal e de relacionamento.

2. Dados coletados

Coletamos dados de cadastro (nome, e-mail, telefone), dados de entrega (endereco completo e CEP), dados de pedido e historico de interacao no site.

3. Finalidades e bases legais (LGPD)

Tratamos dados para executar contratos de compra, cumprir obrigacoes legais/fiscais, prevenir fraude, atender solicitacoes e melhorar o servico.

4. Compartilhamento

Compartilhamos dados estritamente necessarios com gateways de pagamento, transportadoras, provedores de infraestrutura e autoridades quando exigido por lei.

5. Retencao de dados

Dados de pedidos e documentos fiscais podem ser mantidos pelo prazo legal aplicavel (incluindo obrigacoes tributarias). Dados de suporte e contato sao retidos pelo periodo necessario para atendimento e defesa de direitos.

6. Direitos do titular

Voce pode solicitar confirmacao de tratamento, acesso, correcao, anonimização, portabilidade, revogacao de consentimento e eliminacao de dados quando cabivel legalmente.

7. Seguranca

Adotamos controles tecnicos e organizacionais: autenticacao, controle de acesso, monitoramento, trilhas de auditoria e protecoes contra abuso.

8. Contato e canal LGPD

Para solicitacoes de privacidade e LGPD, use nossos canais oficiais em Contato e informe o assunto “Privacidade/LGPD”.

SyraID Access

Entre com autenticação TOTP para desbloquear o ecossistema Syra e acessar o SyraDash.

Use um app autenticador (Google Authenticator, Authy, 1Password...) para gerar o token de 6 dígitos. Cada código é válido por 30 segundos.

O SyraID utiliza TOTP com QR Code. Gere o QR, escaneie com o autenticador e depois insira o código de 6 dígitos para concluir.

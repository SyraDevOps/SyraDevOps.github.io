Como Apagar um Dispositivo (Avançado)

A exclusão de um dispositivo é uma operação de segurança que geralmente requer a execução de um comando HTTP DELETE diretamente no terminal (CMD ou PowerShell no Windows, Terminal no Linux/macOS). Isso garante que a remoção seja intencional e não acidental. Para apagar o dispositivo chamado "teste", execute o seguinte comando: curl -X DELETE "http://Syra-Home-SYRA0156326.local/dispositivos?name=teste"

Nota: Substitua "teste" pelo nome exato do dispositivo que você deseja remover. O comando deve ser executado em uma rede onde o endereço Syra-Home-SYRA0156326.local seja acessível.

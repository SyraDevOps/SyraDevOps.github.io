SyraX — projetos para a comunidade

SyraX reúne projetos abertos desenvolvidos pela comunidade e para a comunidade. São ferramentas, experimentos e kits pensados para educação, pesquisa e integração com seus próprios projetos. Todos os repositórios estão hospedados em nosso GitHub — venha ver, usar e, se quiser, contribuir!

Por que participar?
  • Aprendizado prático com código aberto.
  • Projetos reutilizáveis para escolas e pesquisas.
  • Comunidade ativa: issues, PRs e discussões.

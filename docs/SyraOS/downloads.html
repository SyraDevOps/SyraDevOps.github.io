Instruções de instalação

Windows

1. Baixe o arquivo .exe
2. Execute o instalador
3. Siga o assistente de instalação
4. Inicie a aplicação e faça login
5. Configure sua API key

Linux

1. Baixe o arquivo .tar.gz
2. Execute: tar -xzf arquivo.tar.gz
3. Entre na pasta extraída
4. Execute: ./syra-cloud
5. Acesse via http://localhost:3000

Android

1. Baixe o arquivo APK
2. Abra o gerenciador de arquivos
3. Localize o arquivo APK
4. Toque para instalar
5. Abra a aplicação após instalar

Documentação técnica

SDKs SyraOS (v2.2.0)

Integre a Syra Cloud em sua aplicação. SDKs prontos para TypeScript, JavaScript e Python com token obrigatório para todas as requisições.

TypeScript

syraos.ts | Totalmente tipado

Type-safe, IDE autocomplete, build query avançado com arrays e nested objects.

const syra = createSyra({ 
  token: "sk_live_xxxxx" 
});
await syra.chat({ message: "..." })
Download .ts

JavaScript

syraos.js | ES6 moderno

Compatível com Node.js e browsers, Proxy dinâmico, zero dependências.

import { createSyra } from "syraos";
const syra = createSyra({ 
  token: "sk_live_xxxxx" 
});
await syra.chat({ ... })
Download .js

Python

syraos.py | Pythônico

Orientado a objetos, suporte a context manager, build query com urllib.

from syraos import Syra
api = Syra(token="sk_live_xxxxx")
res = api.chat({ 
  "message": "..." 
})
Download .py

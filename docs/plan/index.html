Sobre o Projeto

O Planetário Itinerante é uma proposta que visa divulgar as Ciências Naturais utilizando como norte a Astronomia, fazendo uso de temáticas ambientais para possibilitar a interdisciplinaridade.

Em suas visitas às comunidades da região do Campo das Vertentes/MG, o Planetário Itinerante oferece oficinas como forma de cumprir o seu objetivo de difusão científica e educação ambiental integrada.
